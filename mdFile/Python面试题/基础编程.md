2、进制转换

```python
i = 100
# 十进制转换为二进制
print(bin(i))
# 不要0b开头
print('{0:b}'.format(i))

# 十进制转换为八进制
print(oct(i))
# 不要0o开头
print('{0:o}'.format(i))

# 十进制转换为16进制
print(hex(i))
#  不要0x开头
print("{0:x}".format(i))


# 二进制转十进制
b = '0b1100100'
print(int(b,2))

# 八进制转十进制
o = '0o144'
print(int(o,8))

# 16进制转10进制
h = '0x64'
print(int(h,16))
```

3、正则匹配输出中文

```python
'''要求只输出中文,不要英文
输出个格式为 
这个是 小李子 二货'''
import re

str1 = 'this is 这个是 name 小李子, and his 二货 !'

res = re.findall(u'[\u4e00-\u9fa5]+',str1)
print(res)

'''
中文在unicode的范围是\u4e00-\u9fa5 要在前面加u说明是unicode编码
'''
```



4、字典排序

```python
dic = {'a':6,'c':4, 'd':5, 'b':7, 'e':2, 'h':1}

# 按key排序
dic_key = sorted(dic.items(), key = lambda i:i[0])
# 按value排序
dic_value = sorted(dic.items(), key = lambda i:i[1])
print(dic_key)
print(dic_value)

```

[('h', 1), ('e', 2), ('c', 4), ('d', 5), ('a', 6), ('b', 7)]
